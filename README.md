# 🧊 NodeBB Unified Dashboard (מרכז הפורומים)

מערכת מתקדמת למשתמשי Tampermonkey המאחדת את כל הפורומים מבוססי NodeBB (כגון מתמחים טופ, תחומים ועוד) לממשק אחד נקי ונוח.

הסקריפט "מזריק" את עצמו לתוך הפורום שבו אתם גולשים, ויוצר דף מרוכז המציג את כל הנושאים שלא נקראו מכל האתרים המחוברים, בעיצוב תואם למקור וללא צורך במעבר בין טאבים.

## ✨ פיצ'רים עיקריים

*   **איחוד נושאים:** צפייה בכל הנושאים שלא נקראו מכל הפורומים ברשימה אחת ממוינת.
*   **אינטגרציה מושלמת:** מוסיף כפתור "מרכז הפורומים" לתפריט הצד/עליון של הפורום הקיים.
*   **גילוי אוטומטי:** בעת כניסה לפורום NodeBB חדש, המערכת תציע להוסיף אותו לרשימה או להתעלם ממנו.
*   **עיצוב הרמוני (Harmony):** הדשבורד מעוצב בהשראת ערכת הנושא הנפוצה, כולל תצוגת אווטארים, אייקונים של קטגוריות ותצוגה מקדימה (Teaser) של התגובה האחרונה.
*   **עקיפת חסימות תמונה (ORB/CORS):** מנגנון חכם לטעינת תמונות פרופיל ולוגואים מאתרים אחרים גם אם יש חסימות אבטחה בדפדפן.
*   **ניהול חכם:** הגדרות מובנות להוספה/הסרה של אתרים ושינוי שמות התצוגה שלהם.
*   **עדכונים אוטומטיים:** הסקריפט מסונכרן מול GitHub ויעדכן את עצמו כשיצאו שיפורים.

## 🚀 התקנה

1.  ודאו שיש לכם תוסף לניהול סקריפטים מותקן בדפדפן (כגון [Tampermonkey](https://www.tampermonkey.net/) או Violentmonkey).
2.  לחצו על הקישור להתקנת הסקריפט:
    **[התקן את NodeBB Dashboard](https://github.com/The-Young-boy/nodebb-dashboard/raw/refs/heads/main/nodebb-dashboard.user.js)**
3.  אשרו את ההתקנה בתוסף.

## ⚙️ שימוש

1.  היכנסו לאחד הפורומים ברשימה (למשל `mitmachim.top`).
2.  חפשו בתפריט הניווט (בצד או למעלה) את הכפתור החדש: **"מרכז הפורומים"**.
3.  לחיצה עליו תפתח את הדשבורד במרכז המסך.
4.  **הוספת אתרים:** 
    *   גילשו לפורום אחר. יקפוץ חלון שישאל אם להוסיף אותו.
    *   לחלופין, פתחו את הדשבורד -> הגדרות -> והוסיפו ידנית.

## 🛠️ דרישות טכניות

*   דפדפן מודרני (Chrome, Edge, Firefox).
*   חיבור פעיל (Login) בכל הפורומים שמהם תרצו למשוך מידע (הסקריפט משתמש ב-Cookies של הדפדפן).
